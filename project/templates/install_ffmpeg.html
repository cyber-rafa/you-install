Como Instalar o FFmpeg

O FFmpeg é um software gratuito e de código aberto usado para processar arquivos multimídia. É necessário para baixar vídeos do YouTube em alta resolução (acima de 720p) e combiná-los com áudio de alta qualidade.

1 Baixar o FFmpeg para Windows

Existem duas formas principais de instalar o FFmpeg:

Opção 1: Baixar a versão pré-compilada (Recomendado)

Baixe a versão mais recente do FFmpeg do site oficial.

Baixar FFmpeg
Opção 2: Usar o gerenciador de pacotes Chocolatey

Se você já tem o Chocolatey instalado, pode usar este método.

choco install ffmpeg
2 Extrair o arquivo baixado (para Opção 1)

Depois de baixar o arquivo .7z, você precisa extraí-lo. Para isso, você pode usar o 7-Zip ou outro programa de descompactação.

  1. Clique com o botão direito no arquivo baixado
  2. Selecione "Extrair aqui" ou "Extrair para ffmpeg\"
  3. Aguarde a conclusão da extração
3 Adicionar o FFmpeg ao PATH do sistema

Para que o aplicativo possa encontrar o FFmpeg, você precisa adicioná-lo à variável de ambiente PATH do Windows:

  1. Copie o caminho completo para a pasta bin dentro da pasta do FFmpeg extraída
    (exemplo: C:\ffmpeg\bin)
  2. Clique no botão Iniciar e pesquise por "Variáveis de ambiente" ou "Edit environment variables"
  3. Clique em "Editar as variáveis de ambiente do sistema"
  4. Na janela que abrir, clique no botão "Variáveis de Ambiente"
  5. Na seção "Variáveis do sistema", encontre a variável "Path" e selecione-a
  6. Clique em "Editar"
  7. Clique em "Novo" e cole o caminho para a pasta bin do FFmpeg
  8. Clique em "OK" em todas as janelas para salvar as alterações
4 Verificar a instalação

Para verificar se o FFmpeg foi instalado corretamente:

  1. Abra um novo Prompt de Comando ou PowerShell (é importante ser uma nova janela)
  2. Digite o seguinte comando e pressione Enter:
ffmpeg -version

Se você ver informações sobre a versão do FFmpeg, a instalação foi bem-sucedida!

5 Reiniciar o aplicativo

Após instalar o FFmpeg e adicioná-lo ao PATH:

  1. Feche o navegador onde o aplicativo de download está aberto
  2. Reabra o aplicativo
  3. Você verá um indicador mostrando que o FFmpeg está disponível
  4. Agora você poderá baixar vídeos em resolução de até 4K!
6 Solução de problemas comuns

Se você estiver tendo problemas:

  • O comando "ffmpeg" não é reconhecido: Verifique se você adicionou o caminho correto ao PATH e reinicie o computador.
  • O aplicativo ainda não reconhece o FFmpeg: Reinicie o aplicativo ou seu navegador.
  • Erros durante o download: Verifique se o FFmpeg foi instalado corretamente executando o comando ffmpeg -version.
