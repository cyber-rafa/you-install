# YouTube Downloader

Aplicativo para baixar vídeos e playlists do YouTube em múltiplas resoluções.

## Funcionalidades

- Suporte para vídeos em até 4K
- Download de playlists completas
- Interface web amigável
- Instalação automática do FFmpeg

## Instalação

```
pip install -r requirements.txt
python project/app.py
```

## Como Usar

1. Acesse http://localhost:5000
2. Cole a URL do vídeo ou playlist
3. Selecione a qualidade e baixe

## Requisitos

- Python 3.6+
- FFmpeg (instalado automaticamente)
- Conexão com a Internet

## Observações

Este aplicativo é para fins educacionais.
